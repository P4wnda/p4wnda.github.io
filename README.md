# p4wnda.github.io

Root landing page for [p4wnda.github.io](https://p4wnda.github.io/).

This repository is intentionally small. It only serves the project hub at the domain root and links to the separate GitHub Pages project sites:

- [Blog](https://p4wnda.github.io/blog/)
- [Pentest Cheatsheet](https://p4wnda.github.io/pentest/)

## Structure

```text
index.html    Landing page
landing.css   Landing page styles
```

The blog and pentest tool live in their own repositories so they can be maintained independently.

## Deploy

GitHub Pages should be configured as:

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
